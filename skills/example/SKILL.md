# Example Skill

This is a placeholder skill demonstrating the standard format.

## Format Requirements

- **Frontmatter**: YAML between `---` delimiters (max 1024 chars)
- **name**: lowercase-kebab-case, letters/numbers/hyphens only
- **description**: starts with "Use when..." to indicate triggering condition
- **Body**: Markdown content with instructions for the agent
